Golanger is a lightweight framework for writing web applications in Go. 


## 框架简单实现了MVC的设计方式

### Features
 * Routing
 * Controllers
 * Templates
 * Session
 * Plugins


### 约定的命名规则：

    Model: 存放在models目录中
    Controller: 存放在controllers目录中
    View: 存放在views目录中
    静态文件放在static目录中
